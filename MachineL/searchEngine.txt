

五个表
1. word 记录每个出现过的单词
2. url 记录每个出现的url
3. link 二个url，一个是from 一个是to。记录url之间的链接关系
4. link word ，一个word id 一个 link id  记录link出现的word.
    这样比如我们在搜索中输入一个word，可能的link就容易找到。
5. word location , url id， word id， location. 每个单词出现在了哪个url的哪个地方。

这样的设计可以使搜索engine很快的得到所以相关的网页链接。
但是按照什么顺序返回给用户是一个问题。就是排名 rank

1. 基于内容的排名
  单词频率 出现了好多次
   单词位置  出现在前面，或者标题，关联性会更大
  单词位置  如果搜索多个单词，位置越近就会越匹配

2. 基于外面的链接

3. pagerank
给与每个网页一个重要性参数，这个参数会动态调整。如果一个网页被链接得越多，
那么它的pagerank就会越大。

4. 根据文本内容的相关程度来排名

5. 记录点击行为并把它放在神经网路中训练
